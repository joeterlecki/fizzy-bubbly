Fizzy Bubbly

Architecture

This application is deployed in AWS using serverless technologies. The frontend is deployed with bootstrap using a static s3 bucket.

The assets are then served using CloudFront CDN and ACM Certificate Management for SSL.

The application logic is deployed with API Gateway and Lambda Proxy Integration with a Python backend.

The Fizzy Bubbly application logic sends a GET request through an API Gateway endpoint with the supplied numbers and then the backend Lambda executes the event with the queryStringParameters to return a number list to the DOM and displayed using some Javascript.
